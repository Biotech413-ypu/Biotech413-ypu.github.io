實驗步驟

  • 樣品處理
      　
    1. 將樣品瓶中加入約1 c.c.的75%酒精
    2. 用吸管中空部分在蔬果上押出約5片圓片,作為蔬果樣品
    3. 將圓片的蔬果樣品放入樣品瓶中
    4. 將含有蔬果的樣品瓶搖晃數秒後靜置 3 分鐘
  • 光源調整(測試片做農藥測試和這個)
    1. 取出兩根檢測管放入白色檢測座中
    2. 樣品組的檢測管加入3-4滴做好的酒精蔬果液(50% 農藥測試改滴75%酒精)
    3. 將手機架好並使其紅框對準白色檢測座中的兩根檢測管
    4. 樣品組紅框對準滴入酒精蔬果液的檢測管；空白組對準未滴入任何樣品的檢測管
    5. 按下光源訊號確認空白組與樣品組的第三個數值(B)大於150且相差不大於15
  • 農藥測試(測試片做光源調整和這個)
    1. 在酵素瓶中加水至畫線處(
    2. 取出一根酵素棒，有凹槽的朝下，放進酵素瓶裡搖晃混合成酵素液
    3. 將兩個檢測管中分別加入一半酵素液(加入時不要再移動檢測座和手機)
    4. 按下開始分析並等待結果
